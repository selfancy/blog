## github 实用统计图标

以 [`openjdk`](https://github.com/openjdk/jdk) 为例, 实时查看仓库统计信息

### 实时 watchers 数
![stars](https://badgen.net/github/watchers/openjdk/jdk)
```
https://badgen.net/github/watchers/<github用户名>/<仓库名>
```
